AseViant

Trabajo solicitado

La Asociación Solidarista de Empleados de AseViant requería un cambio de rediseño, debido al cambio de nombre de AseMedplast a Aseviant

Características del nuevo sitio web

Dado que los usuarios principales del sitio web son los asociados de AseViant, se diseño el sitio web para un fácil acceso a los temas más buscados por los asociados, eliminando imagenes o textos innecesarios:

  • Usabilidad del sitio, pensando en el usuario final (los asociados)
  • Una navegación más sencilla y amigable para el visitante
  • Carga rápida del sitio web (menos de 2s)
  • Sitio optimizado para dispositivos móviles

Paquete Solicitado

Plan Poas, de 20 páginas. Se agregaron páginas adicionales.
