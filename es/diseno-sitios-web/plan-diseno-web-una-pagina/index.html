Plan onePage

Es un sitio web de dos (2) páginas con todas las funcionalidades de un diseño tradicional. No posee límite de texto aunque se recomienda publicar la información más relevante para no confundir al usuario.

Nuestro Plan onePage es ideal para PyMEs, Profesionales, Trabajadores Independientes, Instituciones y en general para los que que deseen tener presencia en la Internet con una inversión baja, pero de gran calidad y funcionalidad. La estructura normal de este plan es la siguiente:

  1. Página principal:
    Información de la empresa, lista de servicios o productos ofrecidos y mapa de Google de ubicación
  2. Página de contacto:
    Información de contacto y formulario de contacto

Características del Plan onePage

Características generales

  • Diseño web personalizado. No utilizamos plantillas comerciales
  • Diseño web amigable para dispositivos móviles
  • Diseño web en cumplimiento de estándares de diseño web internacionales
  • Registro de dominio gratuito (.com, .net, .biz, .org)
  • Hospedaje web
  • Diseño de logo básico
  • Posicionamiento del sitio con SEO básico - optimización del motor de búsqueda

Características específicas

  • 2 páginas o secciones
  • 1 formulario de contacto
  • 1 actualización mensual del contenido (texto e imagenes)
  • 4 imagenes de alta definición gratuitas
  • 2 cuentas de correo electrónico @sudominio.com
  • Tiempo de entrega: 7 - 15 días laborales
